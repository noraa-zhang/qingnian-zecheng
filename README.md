# 青年择城通 (CityMatch)

面向毕业生/海归/创业者的城市补贴AI决策工具。64城×3身份政府人才补贴数据库，选身份、选偏好、出排名，每条数据锚定官方政策原文。

**🌐 在线访问**：https://noraa-zhang.github.io/qingnian-zecheng/

---

## 功能

- **浏览城市**：按地域/身份筛选64城补贴卡片，含补贴金额、申领条件、企业数量
- **智能匹配 — 双输入模式**：
  - 自然语言输入：用户自由描述偏好 → Prompt Chain（LLM 槽位提取）→ 硬性过滤 + 双因子归一化排序 → 可追溯的个性化城市排名
  - 结构化选择：6道选择题 → 同上过滤+排序链路
- **收藏夹**：保存意向城市，汇总补贴总额

## 数据规模

| 维度 | 规模 |
|------|------|
| 城市 | 64城（一线/新一线/二线全覆盖） |
| 身份 | 3种（应届毕业生/留学归国/创业者） |
| 数据点 | 192条 |
| 政策链接 | 每条锚定政府官方原文（62城已逐条核验） |
| 资格字段 | 56城已摘录age_max/social_security/requires_hukou |
| 企业数据 | 64城×4类（国企/外企/民企/创业公司） |

## 推荐链路

三阶段设计：

```
自然语言输入（自由文本）
  → Prompt Chain（LLM 意图识别 + 槽位提取 → 结构化查询参数）
  → 硬性过滤（地域 + 年龄上限 + 补贴可用性三重校验）
  → 候选池内双因子归一化排序
     ├─ 补贴金额得分 × 0.7
     └─ 企业数量得分 × 0.3（支持多选类型，取均值）
```

边界兜底：LLM 解析失败→回退结构化选择 / 候选池为空→无匹配提示 / 金额相同(max=min)→满分 / 多选企业类型取均值避免权重失真

## 技术

- 单文件 React 18（CDN）+ Babel Standalone
- CSS 变量主题系统 + 响应式布局
- localStorage 持久化（收藏夹 + 身份偏好）
- GitHub Pages 部署（HTTPS，解决 gov.cn Referer 拦截）
